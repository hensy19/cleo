CLEO – Period Tracking & Wellness Web Application

Overview
CLEO is a modern web-based period tracking and wellness application designed to help users monitor their menstrual cycle, track symptoms, and gain insights into their health. The application focuses on providing a clean, user-friendly interface along with meaningful tracking features to improve overall user experience.

Features

📅 Cycle Tracking – Log and monitor menstrual cycle dates

🤕 Symptom Tracking – Track symptoms like cramps, mood swings, fatigue, etc.

📊 History & Insights – View past records and identify patterns

🎨 User-Friendly UI – Clean and responsive design for better usability

🔐 Data Management – Secure handling of user data

⚡ Real-Time Updates – Instant updates for user inputs
